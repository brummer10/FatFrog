# FatFrog.lv2
High Gain Amplifier

This is a special edition for the [MOD] (https://moddevices.com/)

## Features

- Valve amplifier simulation. 

## Dependencys

- lv2-dev

## Build

- make
- make install # will install into ~/.lv2 ... AND/OR....
- sudo make install # will install into /usr/lib/lv2
